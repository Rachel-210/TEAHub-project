# 🧩 TEAHub / GAMEHub: Uma Ponte Digital para a Terapia do Autismo

## 🚀 Visão Geral do Projeto

O **TEAHub/GAMEHub** é um sistema digital duplo e inovador, concebido para revolucionar o acompanhamento e a intervenção terapêutica em clínicas que atendem crianças com Transtorno do Espectro Autista (TEA).

Nosso objetivo central é otimizar o processo terapêutico, tornando-o **mais objetivo, lúdico e centrado em dados**, transformando o tempo de brincadeira da criança em *insights* clínicos valiosos para os profissionais.

## 🛠️ Componentes Principais

O sistema é dividido em duas interfaces que se integram de forma contínua:

### 1. TEAHub (Sistema de Gestão e Monitoramento)

Voltado para a **Clínica** e o **Profissional**, o TEAHub é um sistema administrativo e analítico.

| Recurso (RF) | Descrição |
| :--- | :--- |
| **Dashboard em Tempo Real** (RF-016) | Visualização instantânea do desempenho da criança durante a sessão do GAMEHub. |
| **Gestão de Prontuários** (RF-012) | Criação, consulta e atualização de prontuários eletrônicos dos pacientes. |
| **Geração de Relatórios** (RF-015) | Relatórios detalhados e personalizados para acompanhamento do desenvolvimento. |
| **Gestão Financeira/Acesso** (RN-001) | Controle de assinaturas mensais e acesso de profissionais da clínica. |

### 2. GAMEHub (Aplicativo Lúdico e Interativo)

É a interface direta com o paciente, focada em engajamento e estímulo cognitivo/motor.

| Recurso (RF) | Descrição |
| :--- | :--- |
| **Criação de Avatar** (RF-022) | Personalização da experiência para maior identificação e engajamento. |
| **Estrutura de Níveis** (RF-024) | Conteúdo organizado em níveis crescentes de dificuldade para estímulo contínuo. |
| **Sistema de Premiação** (RF-025, RN-005) | Recompensas com Moedas Virtuais após a conclusão de níveis, incentivando a progressão. |
| **Acesso Restrito** (RF-020, RN-004) | Login seguro do paciente via CPF validado pela clínica, garantindo rastreabilidade. |
